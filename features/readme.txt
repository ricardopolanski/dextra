### How to clone repository and execute the project ###

1 - Open the VSCode;
2 - On the Welcome Page, press CTRL+Shift+E then click on Clone Repository;
3 - Paste the URL Repository "https://github.com/ricardopolanski/dextra" on the top of the Welcome Page and press Enter;
4 - Create a new folder and select this one as Repository Location;
5 - A message will be showed up and you can choose any option: "Open" or "Open in New Window";
6 - To execute the test open a new Terminal clicking on the Terminal > New Terminal;
7 - Execute the command "cucumber" to start the test.
